Enlaces y navegación en Tablas en HTML: Organiza tu información de forma clara y eficiente

Las tablas son elementos HTML esenciales para presentar información tabular de forma organizada y clara. Permiten estructurar datos en filas y columnas, facilitando su lectura y comprensión.


Estructura de una tabla (table, tr, th, td):

  • table: Define el elemento principal de la tabla.
  • tr: Define una fila de la tabla.
  • th: Define una celda de encabezado de la tabla.
  • td: Define una celda de datos de la tabla.

Ejemplo

<table>
        <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
    </tr>
    <tr>
        <td>Juan Pérez</td>
        <td>30</td>
        <td>Medellín</td>
    </tr>
    <tr>
        <td>María Gómez</td>
        <td>25<
        <td>Bogotá</td>
    </tr>
</table>

Encabezados de tabla (th):

Las celdas de encabezado (th) se utilizan para definir los títulos de las columnas de la tabla.


Ejemplo

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
     </tr>
</table>

Celdas de datos (td):

Las celdas de datos (td) se utilizan para contener el contenido de la tabla.


Ejemplo

<table>
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
        <th>Ciudad</th>
    </tr>
    <tr>
        <td>Juan Pérez</td>
        <td>30</td>
        <td>Medellín</td>
    </tr>
    <tr>
        <td>María Gómez</td>
        <td>25</td>
        <td>Bogotá</td>
    </tr>
 </table>

Atributos de tablas (border, cellspacing, cellpadding):

  • border: Define el borde de la tabla.
  • cellspacing: Define el espacio entre las celdas de la tabla.
  • cellpadding: Define el espacio entre el contenido de la celda y el borde de la celda.

Ejemplo

<table border="1" cellspacing="2" cellpadding="5"></table>
