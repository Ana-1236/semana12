Diseño de página con CSS: Cajas, posicionamiento y diseño flexible.


Cajas de contenido (content-box) y modelo de caja (box-model)
En CSS, el diseño de la página se basa en el concepto de cajas. Cada elemento HTML se representa como una caja rectangular con propiedades como el ancho, la altura, el margen, el relleno y el borde. Estas propiedades se definen utilizando el modelo de caja, que especifica cómo se calculan las dimensiones totales de una caja.


Caja de contenido (content-box)

El modelo de caja de contenido es el modelo predeterminado en CSS. En este modelo, el ancho y la altura de una caja solo incluyen el contenido de la caja (texto, imágenes, etc.). El margen, el relleno y el borde se agregan al ancho y la altura total de la caja, lo que puede generar resultados inesperados si no se tienen en cuenta.



    #caja {
     width: 100px;
     height: 100px;
     border: 10px solid #000;
     padding: 20px;
     }

En este ejemplo, la caja tendrá un ancho total de 140px (100px de ancho + 20px de relleno a cada lado) y una altura total de 140px (100px de altura + 20px de relleno arriba y abajo).


Modelo de caja (box-model)

El modelo de caja alternativo, conocido como modelo de caja "box-sizing: border-box", permite incluir el margen, el relleno y el borde dentro del ancho y la altura de la caja. Esto facilita el cálculo de las dimensiones totales de la caja y evita sorpresas inesperadas.


#caja {
        width: 100px;
        height: 100px;
        border: 10px solid #000;
        padding: 20px;
        box-sizing: border-box;
      }

En este ejemplo, la caja tendrá un ancho total de 100px (ancho especificado) y una altura total de 100px (altura especificada). El borde y el relleno no se agregan al ancho y la altura, ya que se incluyen dentro de las dimensiones de la caja.


Posicionamiento.

CSS proporciona diversas propiedades para posicionar elementos en la página. Algunas de las propiedades de posicionamiento más comunes son:

  • static: El posicionamiento predeterminado, donde el elemento se coloca en el flujo normal del documento.
  • relative: Posiciona el elemento relativo a su posición normal en el flujo del documento. Los cambios de posición no afectan a los elementos circundantes.
  • absolute: Posiciona el elemento de forma absoluta en relación con su contenedor más cercano. Elimina el elemento del flujo normal del documento, lo que permite superponerlo a otros elementos.
  • fixed: Posiciona el elemento de forma fija en relación a la ventana del navegador. El elemento permanece en la misma posición incluso cuando se desplaza la página.
  • flex: Posiciona el elemento dentro de un contenedor flexible, utilizando las propiedades de diseño flexible.
Ejemplo:
#elemento1 {
            position: relative; /* Posicionamiento relativo */
            top: 20px; /* Mueve el elemento 20px hacia arriba */
            left: 50px; /* Mueve el elemento 50px hacia la izquierda */
             }
                  
#elemento2 {
            position: absolute; /* Posicionamiento absoluto */
            top: 100px; /* Posiciona el elemento 100px desde la parte superior */
            left: 100px; /* Posiciona el elemento 100px desde la izquierda */
            }

En este ejemplo, el elemento 1 se posiciona 20px hacia arriba y 50px hacia la izquierda de su posición normal en el flujo del documento. El elemento 2 se posiciona de forma absoluta en la página, a 100px de la parte superior y 100px de la izquierda.


Flotación (float)

La propiedad float permite que un elemento se mueva a un lado del flujo normal del documento, como si flotara sobre el contenido circundante. Otros elementos se envuelven alrededor del elemento flotante. 


#columna1 {
        float: left;
        width: 50%;
        padding: 10px;
          }
                  
#columna2 {
        float: right;
        width: 50%;
        padding: 10px;
          }

En este ejemplo, #columna1 se flotará hacia la izquierda y #columna2 se flotará hacia la derecha, creando un diseño simple de dos columnas.


Flexbox

Flexbox es un modelo de diseño más moderno y flexible que permite a los desarrolladores alinear elementos a lo largo de un eje principal (horizontal o vertical) y distribuir el espacio restante de manera uniforme. Flexbox ofrece un mayor control sobre la disposición de los elementos. 


.contenedor {
            display: flex; /* Habilita flexbox para el contenedor */
            flex-direction: row; /* Establece la dirección del eje principal (horizontal) */
            justify-content: space-between; /* Distribuye el espacio restante entre elementos */
            align-items: center; /* Alinea los elementos verticalmente al centro */
             }
                  
.columna {
        flex: 1; /* Le da a cada elemento el mismo tamaño flexible */
        padding: 10px;
         }

En este ejemplo, el contenedor utiliza flexbox para disponer los elementos columna en una fila (horizontal) con espacio distribuido entre ellos y alineados verticalmente al centro.
