Elementos multimedia en HTML: Dale vida a tu contenido web


Los elementos multimedia son esenciales para enriquecer el contenido de una página web y brindar una experiencia más atractiva e interactiva a los usuarios. HTML ofrece diversas etiquetas para insertar imágenes, videos, audio y objetos incrustados en tus páginas web.


Imágenes (img):

La etiqueta <img> se utiliza para insertar imágenes en una página web. Permite definir la ruta de la imagen, su tamaño, texto alternativo y otras características.


Ejemplo

<img src="imagen.jpg" alt="Descripción de la imagen" width="200" height="150">

Videos (video):

La etiqueta <video> se utiliza para insertar videos en una página web. Permite definir la ruta del video, sus controles de reproducción y otras características.


Ejemplo

<video src="video.mp4" controls width="640" height="360">
    <p>Tu navegador no admite videos HTML5.</p>
</video>

Audio (audio):

La etiqueta <audio> se utiliza para insertar audio en una página web. Permite definir la ruta del archivo de audio, sus controles de reproducción y otras características.


Ejemplo


        <audio src="audio.mp3" controls>
            <p>Tu navegador no admite audio HTML5.</p>
        </audio>

Objetos incrustados (embed, iframe):

Las etiquetas <embed> e <iframe> se utilizan para insertar contenido multimedia de terceros, como videos de YouTube, mapas de Google o presentaciones de Slideshare.


Ejemplo


        <iframe src="https://www.youtube.com/watch?v=ID_VIDEO" width="560" height="315"></iframe>
