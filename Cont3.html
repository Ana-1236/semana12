Elementos de estructura y organización en HTML: Crea páginas web con significado


HTML (HyperText Markup Language) es el lenguaje fundamental para crear páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web, utilizando etiquetas y atributos que indican cómo debe ser interpretada por los navegadores web.


Secciones (section):

La etiqueta <section> se utiliza para definir secciones temáticas dentro de una página web. Permite agrupar contenido relacionado y darle un significado semántico específico.


Ejemplo

  <section class="noticias">
    <h2>Últimas noticias</h2>
        <article>
            <h3>Título de la noticia 1</h3>
            <p>Contenido de la noticia 1...</p>
        </article>
        <article>
            <h3>Título de la noticia 2</h3>
            <p>Contenido de la noticia 2...</p>
         </article>
 </section>
 <section class="contactos">
        <h2>Información de contacto</h2>
        <p>Dirección: ...</p>
        <p>Teléfono: ...</p>
        <p>Correo electrónico: ...</p>
 </section>

Artículos (article):

La etiqueta <article> se utiliza para definir contenido independiente y autosuficiente dentro de una página web, como noticias, entradas de blog o publicaciones.


Ejemplo

<article>
    <h2>Título del artículo</h2>
    <p>Contenido del artículo...</p>
    <footer>
        <p>Autor: ...</p>
        <p>Fecha de publicación: ...</p>
     </footer>
</article>

Encabezado (header):

La etiqueta <header> define la sección del encabezado de una página web, que generalmente contiene el logotipo, el menú de navegación y otros elementos importantes.


Ejemplo

<header>
    <img src="logo.png" alt="Logotipo">
    <nav>
        <ul>
            <li><a href="#">Inicio</a></li>
            <li><a href="#">Acerca de</a></li>
            <li><a href="#">Contacto</a></li>
        </ul>
    </nav>
</header>

Pie de página (footer):

La etiqueta <footer> define la sección del pie de página de una página web, que generalmente contiene información de copyright, enlaces a redes sociales y otros datos relevantes.


Ejemplo

<footer>
    <p> © 2024 Todos los derechos reservados.</p>
    <ul>
         <li><a href="#"><i class="fab fa-facebook"></i></a></li>
        <li><a href="#"><i class="fab fa-twitter"></i></a></li>
        <li><a href<i class="fab fa-instagram"></i></a></li>
    </ul>
</footer>
