Actividad Final.

Actividad: Crea tu propia página web usando HTML


Objetivo: El objetivo de esta actividad es crear una página web básica utilizando los fundamentos del lenguaje HTML.


    Crearás una página web que incluya las siguientes secciones:


  • Título: Un título descriptivo que se mostrará en la barra del navegador.
  • Encabezado: Un encabezado principal que presente el tema de tu página web.
  • Secciones: La página web estará dividida en secciones temáticas utilizando etiquetas HTML apropiadas.
  • Contenido: Cada sección incluirá contenido relevante al tema, como párrafos, listas, imágenes y enlaces.
  • Pie de página: Un pie de página con información adicional, como tu nombre y copyright.

Requisitos: Utilizarás HTML para estructurar el contenido de tu página web.


    Pasos:

  • Planificación: Decide el tema de tu página web y el contenido que deseas incluir. Haz un boceto o lista para organizar las secciones y el contenido.
  • Estructura básica: Crea un archivo HTML básico con la etiqueta <!DOCTYPE html>, la etiqueta <html>, el elemento <head> y el elemento <body>.
  • Título: La Define el título de la página web utilizando la etiqueta <title> dentro del elemento <head>.
  • Encabezado: Crea un encabezado principal utilizando la etiqueta <h1> dentro del elemento <body>.
  • Secciones: Divide el contenido de tu página web en secciones utilizando etiquetas HTML como:
    • <h2> - <h6> para subtítulos
    • <p> para párrafos
    • <ul> y <ol> para listas ordenadas y desordenadas
    • <img> para incluir imágenes
    • <a> para crear enlaces a otras páginas web o secciones de la misma página (anclas)
  • Pie de página: Crea un pie de página utilizando la etiqueta <footer> e incluye información adicional como tu nombre y copyright.
