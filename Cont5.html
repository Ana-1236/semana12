Enlaces y navegación en HTML: Conecta tu contenido y facilita la exploración

Los elementos de enlaces y navegación son esenciales para crear una estructura lógica y facilitar la exploración de un sitio web. HTML ofrece diversas etiquetas para definir enlaces internos, externos, de correo electrónico, anclas y menús de navegación.


Enlaces internos (a):

La etiqueta <a> se utiliza para crear enlaces internos entre diferentes secciones de una misma página web o entre páginas distintas del mismo sitio.


Ejemplo

<a href="#seccion2">Ir a la Sección 2</a>

Enlaces externos (a):

La etiqueta <a> se utiliza para crear enlaces externos a páginas web de otros sitios.


Ejemplo

<a href="https://www.google.com/">Ir a Google</a>

Enlaces de correo electrónico (mailto):

La etiqueta лa> se utiliza para crear enlaces que permitan enviar un correo electrónico a una dirección específica.


Ejemplo

<a href="mailto:correo@ejemplo.com">Enviar correo electrónico</a>

Anclas (id, name)::

Las anclas permiten crear marcadores dentro de una página web para facilitar la navegación a secciones específicas. Se definen utilizando las etiquetas <id> o <name> y se asocian a enlaces mediante el atributo href.


Ejemplo


        <h2 id="seccion2">Sección 2</h2>
        <a href="#seccion2">Ir a la Sección 2</a>

Menús de navegación (nav):

La etiqueta <nav> se utiliza para definir la sección de navegación principal de una página web, que generalmente contiene una lista de enlaces a diferentes secciones del sitio.


Ejemplo


    <nav>
        <ul>
            <li><a href="#">Inicio</a></li>
            <li><a href="#">Acerca de</a></li>
            <li><a href="#">Contacto</a></li>
        </ul>
    </nav>
