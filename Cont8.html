Semántica HTML: Estructura tu contenido de forma significativa

La semántica HTML es esencial para crear páginas web que sean comprensibles tanto para los usuarios como para los motores de búsqueda. Los elementos y atributos semánticos proporcionan información sobre el significado del contenido de la página, lo que ayuda a los usuarios a navegar y comprender mejor la información, y a los motores de búsqueda a indexar y clasificar correctamente la página.


Nuevos elementos semánticos (header, nav, article):

HTML5 introdujo nuevos elementos semánticos que definen claramente el tipo de contenido que contienen. Estos elementos ayudan a estructurar mejor el contenido de la página y a mejorar la accesibilidad.


  • header: Define el encabezado de la página, que generalmente contiene el logotipo, el título de la página y la barra de navegación.
  • nav: Define la sección de navegación principal de la página, que generalmente contiene enlaces a las diferentes secciones del sitio web.
  • article: Define un artículo independiente, como una entrada de blog o un post de noticias. 
<header>
    <h1>Mi Blog</h1>
    <nav>
        <ul>
            <li><a href="#">Inicio</a></li>
            <li><a href="#">Acerca de</a></li>
            <li><a href="#">Contacto</a></li>
        </ul>
    </nav>
</header>
                  
<article>
    <h2>Título del artículo</h2>
    <p>Contenido del artículo...</p>
</article>

Atributos de datos (data-*):

Los atributos de datos (data-*) permiten almacenar información personalizada en los elementos HTML sin afectar a la estructura o al estilo de la página. Esta información puede ser utilizada por scripts JavaScript o aplicaciones web para obtener datos adicionales sobre el elemento. 


<div data-id="123" data-nombre="Juan Pérez">
    </div>

Microdatos (microdata):

Los microdatos son una forma de estructurar datos dentro de elementos HTML de una manera que sea comprensible para los motores de búsqueda y otras aplicaciones. Los microdatos utilizan vocabularios predefinidos o personalizados para definir entidades y sus propiedades. 


<div itemscope itemtype="http://schema.org/Person">
    <h2 itemprop="name">Juan Pérez</h2>
    <p itemprop="email">juan.perez@ejemplo.com</p>
</div>
