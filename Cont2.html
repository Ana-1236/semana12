Encabezados (<h1 > a <h6 >):

Los encabezados son elementos esenciales para organizar el contenido de una página web y facilitar su lectura. Se definen utilizando etiquetas de apertura y cierre, y se clasifican en seis niveles (h1 es el más importante, h6 el menos importante).


Ejemplo

 
            <h1>Título principal</h1>
            <h2>Título secundario</h2>
            <h3>Título terciario</h3>
            <h4>Título de cuarto nivel</h4>
            <h5>Título de quinto nivel</h5>
            <h6>Título de sexto nivel</h6>

Párrafos (<p>):

Los párrafos son los bloques de texto que forman el cuerpo principal del contenido de una página web. Se definen utilizando la etiqueta <p>.


Ejemplo

 <p>Este es un párrafo de texto. Puedes utilizar esta etiqueta para crear bloques de texto que formen el cuerpo principal del contenido de tu página web.</p >

Formato de texto (negrita, cursiva, subrayado):

Para aplicar formato de texto a elementos específicos dentro de un párrafo, se utilizan las etiquetas <b> (negrita), <i> (cursiva) y <u> (subrayado).


Ejemplo

Esta frase está en <b>negrita</b>, <i>cursiva</i> y <u>subrayada</u>.

Listas (<ul>, <ol>, <li>):

Las listas son una forma organizada de presentar información en una página web. Se definen utilizando las etiquetas <ul> (lista sin orden), <ol> (lista ordenada) y <li> (elemento de la lista).


Ejemplo

<ul<
    <li>Elemento 1 de la lista sin orden </li>
     <li>Elemento 2 de la lista sin orden</li>
     <li>Elemento 3 de la lista sin orden</li>
</ul>
<ol>
     <li>Primer paso</li>
     <li>Segundo paso</li>
     <li>Tercer paso</li>
</ol>

Bloques de código (pre):

Los bloques de código se utilizan para mostrar fragmentos de código fuente o texto preformateado en una página web. Se definen utilizando la etiqueta <pre>.


Ejemplo

<pre>
        <h1>Este es un título</h1>
        <p>Este es un párrafo de texto.</p>
 </pre>
