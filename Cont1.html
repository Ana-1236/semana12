Introducción a HTML: Dominando el lenguaje fundamental de la web.


¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje fundamental para crear páginas web. Es un lenguaje de marcado que define la estructura y el contenido de una página web, utilizando etiquetas y atributos que indican cómo debe ser interpretada por los navegadores web.


Estructura básica de un elemento HTML

Un documento HTML básico tiene la siguiente estructura:

 
                    <!DOCTYPE html>
                    <html>
                    <head>
            <title>Título de la página</title>
              </head>
            <body>
            </body>
              </html>
  • !DOCTYPE html: Declaración de tipo de documento que indica al navegador que se trata de un documento HTML5.
  • html: Etiqueta raíz que contiene todo el contenido del documento HTML.
  • head: Etiqueta que define el título de la página web, que se muestra en la barra de título del navegador.
  • body: Sección que contiene el contenido visible de la página web, como texto, imágenes, enlaces y otros elementos.

Elementos HTML básicos

Los elementos HTML son los componentes básicos que forman el contenido de una página web. Se definen utilizando etiquetas de apertura y cierre, y pueden contener atributos que modifican su comportamiento. Algunos ejemplos de elementos HTML básicos son:

  • <h1> a <h6>: Etiqueta para crear un hipervínculo a otra página web o a una sección específica de la misma página.
  • <p>: Etiqueta para definir un párrafo de texto.
  • <img>: Etiqueta para insertar una imagen en la página.
  • <ul>: Etiqueta para crear una lista sin orden.
  • <ol>: Etiqueta para crear una lista ordenada.
  • <li>: Etiqueta para definir un elemento de una lista.
  • <table> >: Etiqueta para crear una tabla.
  • <tr>: Etiqueta para definir una fila de una tabla.
  • <td>: Etiqueta para definir una celda de una tabla.

Atributos HTML

Los atributos HTML se utilizan para modificar el comportamiento o la apariencia de los elementos HTML. Se escriben dentro de la etiqueta de apertura del elemento, y se componen de un nombre y un valor separados por un signo igual. Por ejemplo, para establecer el color del texto de un párrafo en rojo, se utilizaría el siguiente código: 

 <p style="color: red;">Este texto es rojo</p >
